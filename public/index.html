Frugal IoT project

Frugal IoT is an open-source project to make sensors affordable and accessible to a wide range of projects in resource-constrained situations.

This is a place holder while the page is developed - but there is plenty already online.

See:

  • Live dashboard for Dev projects - you will need to register to access it.
  • Documentation wiki
  • Github repos for:
    • Server,
    • Browser or Mobile Client,
    • MQTT logger,
    • Sensor Nodes
